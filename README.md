# Agenda de contactos
Aplicacion web CRUD diseñada con Laravel y MySQL.

## Versiones utilizadas
- PHP: 8.2.12
- Laravel: 12.59.0
- Composer: 2.9.8
- Node: 24.15.0

## Instalación
``` bash
git clone 
cd
composer install
cp .env.example .env
php artisan key:generate
php artisan migrate --seed
php artisan serve
```
